# @som/archiver-cli

very easy to use

## Installation

```sh
# global cli
npm i -g serverless-config-generator
yarn global add serverless-config-generator

//or

npm i -D serverless-config-generator
yarn add -D serverless-config-generator
```

## How to use

if you install globally
you could run

```sh
# get zip
node-zip <path>
# get tar.gz
node-tar <path>
```

or

```js
const {
  zip,
  tar,
  util,
  factory,
  cli
} = require('@som/archiver-cli')

//write code below
...

```
